# Welcome to Frosts

!["Frost Logo"](images/frosts_logo.svg)

**Frosts** is a lightweight data science framework for Office Scripts.

## 🚀 Features

- pandas-like API for Excel automation
- Merge, group, filter, and more
- Seamless Power Automate integration
